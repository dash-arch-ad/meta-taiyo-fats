## re_mo のキーワード合計について

`re_mo` シートでは、通常のキャンペーン別 Reach 出力に加えて、`APP_SECRET_JSON` の `re_mo_filters` に設定したキーワードごとの合計 Reach も出力しています。

この合計値は、通常のキャンペーン行を単純加算しているのではなく、Meta Insights API に対してキーワードごとのフィルタ条件で別クエリを実行して取得しています。  
そのため、同一キーワードに該当する複数キャンペーンの重複を含む単純合算とは異なる値になります。

設定例:

```json
"re_mo_filters": [
  "XXXX",
  "XXXX",
  "XXXX",
  "yyyy"
]
